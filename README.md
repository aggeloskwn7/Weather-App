# Weather app

Just a nice website made by me with react and tailwind for the frontend and node js for the backend (all in typescript though)

If you wanna see it for yourself download or pull the repo and do npm install and just do npm run dev to check it out on localhost 

https://i.imgur.com/18qyZEw.png

Features: 


Current Weather for any location

Hourly Forecast (up to 48 hours)

5-Day Forecast

Precipitation Map using OpenWeatherMap overlays

Theme Toggle – Light/Dark mode

Unit Switcher – Celsius ↔ Fahrenheit

Detailed Weather Info – humidity, wind, pressure, UV index, and more
